# Summary

I remixed this because using Don1337's carriage http://www.thingiverse.com/thing:2193695 with my V6 clone had the duct almost even with the nozzle tip causing it to occasionally hit the print.  All this does is brings it up a couple mm's to allow clearance.  Arjjck's Center Nozzle Fan has been the best duct to date I have used.  No fancy stuff needed.
